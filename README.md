# BlueDoll

Objective :
  - User Auth (Login - Register)
  - View All Doll (Listview)
  - Share the joy of dolls
  - Check Store Location via Google Map

Database :
  Table User :
    - id (PK)
    - name
    - email
    - password
  Table dolls :
    - id (PK)
    - name 
    - description
    - image

Form :
  - Login Form
  - Register Form
  - View All Doll Form
  - Modify Doll Form (CRUD)
  - View Doll Detail Form
  - View Store
